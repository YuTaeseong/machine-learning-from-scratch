# 선형 회귀

## 회귀란 무엇인가?

영어로 Regression

## 에러와 오차의 차이

에러는 확률 변수
오차는 값

## 모델 결정

y = ax + b;

## 오차 결정

Y = aX + b + e;
e는 에러

e는 확률변수 고로 Y도 확률변수

E(Y) = E(aX+b) + E(e) = aX + b;
d = Y - E(Y) = Y - (aX + b);
Y와 X는 주어짐

> sum(d^2) = sum((Y - (aX + b))^2);

## 파라미터 결정

min(sum(d^2))을 만족하는 a와 b.

a와 b에 대해 각각 편미분한 뒤 0이 되는 지점을 찾자.
